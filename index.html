Knowledge Graph Embedding using TransE

Developed a machine learning model using the TransE algorithm to generate low-dimensional embeddings of entities and relationships in a knowledge graph. This project involved implementing and fine-tuning the algorithm to accurately represent complex graph structures in a continuous vector space. The resulting embeddings improved the efficiency of tasks like link prediction and entity classification, enhancing the overall performance of knowledge graph-based applications.

Cyber bullying detection using Machine Learning

To address the growing issue of cyberbullying, I developed a machine learning model designed to detect harmful online behavior. I started by collecting and preprocessing a dataset of online interactions, labeling instances of cyberbullying. Utilizing Python and natural language processing techniques, I trained various machine learning models to identify abusive content. The resulting model achieved high accuracy in detecting cyberbullying, contributing to enhanced online safety by enabling timely intervention and prevention.

Virtual Mouse using hand gestures

To innovate user interaction, I developed a virtual mouse system controlled by hand gestures. The project aimed to replace traditional mouse input with gesture recognition technology for a more intuitive user experience. I utilized computer vision techniques and machine learning algorithms to track and interpret hand movements in real-time using Python and OpenCV. The system allowed users to perform common mouse actions such as clicking and scrolling through natural hand gestures. As a result, the virtual mouse provided a novel and accessible way to interact with devices, enhancing user convenience and engagement.
